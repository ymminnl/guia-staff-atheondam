Principios Fundamentales

Solo Templates

No puedes crear castigos personalizados. Debes usar exclusivamente las plantillas predefinidas del sistema.

Investiga Primero

Siempre revisa el historial del jugador con /history antes de aplicar cualquier sanción.

Evidencia Obligatoria

Cada sanción debe tener evidencias en Discord (#sanciones-evidencias) con capturas o videos en un hilo.

Procedimiento de Evidencias

  1. Aplicas la sanción en el servidor
  2. El webhook envía mensaje automático a #sanciones-evidencias
  3. TÚ DEBES crear un HILO en ese mensaje
  4. Sube capturas/videos en el hilo como evidencia
  5. El mensaje del webhook + el hilo quedan registrados permanentemente

Procedimiento Completo

Pasos a Seguir

  1. Investiga: /history Jugador + /dupeip Jugador
  2. Toma evidencias: Captura de pantalla o video
  3. Aplica sanción: /ban Jugador Template Razón
  4. Ve a Discord #sanciones-evidencias
  5. Crea HILO en el mensaje del webhook
  6. Sube evidencias en el hilo

No Hacer

  • • NO sancionar sin evidencias
  • • NO usar templates incorrectos
  • • NO abusar del poder
  • • NO olvidar subir evidencias a Discord
  • • NO remover castigos sin consultar a admins

Ejemplos Prácticos

Ejemplo 1: Helper - Jugador haciendo Spam

Situación: Jugador está spammeando en el chat

Pasos:
  1. Verifica historial: /history SpammerPlayer
  2. Si es primera vez, advierte: /warn SpammerPlayer Spam Por favor no hagas spam
  3. Si continúa, congela: /freeze SpammerPlayer
  4. Habla con él para que entienda la regla
  5. Si persiste: /kick SpammerPlayer Spam-Rapido
  6. Reporta a Moderator para que aplique mute si es necesario

Nota: Helper NO puede mutear, solo warn/kick. Para mutes, contactar a Moderator.

Ejemplo 2: Moderator - Jugador haciendo Spam (continuación)

Situación: Helper reporta spam persistente

Pasos:
  1. Revisa el reporte del Helper
  2. Verifica historial: /history SpammerPlayer
  3. Ve evidencias en Discord #sanciones-evidencias
  4. Aplica mute: /mute SpammerPlayer Spam Spam persistente después de warn y kick
  5. Crea hilo en Discord con las evidencias

🔄 Sistema aplica escalera: Primera ofensa = 1h mute

Ejemplo 3: Helper - Sospecha de Fly Hacks

Situación: Observas jugador volando sospechosamente

Pasos:
  1. Activa vanish: /vanish
  2. Teleportate: /tp SuspectPlayer
  3. Observa su comportamiento
  4. Congela si confirmas: /freeze SuspectPlayer
  5. Toma video/capturas como evidencia
  6. Contacta a Moderator inmediatamente
  7. Si es obvio y Helper tiene certeza: /warn SuspectPlayer TrampasLeves

⚠️ Helper NO puede banear. Para bans por hacks, debe contactar Moderator/Admin.

Ejemplo 4: Moderator - Confirmar y Banear Hacker

Situación: Helper reporta hacker con evidencias

Pasos:
  1. Revisa evidencias del Helper (video/capturas)
  2. Verifica historial: /history HackerPlayer
  3. Revisa multicuentas: /dupeip HackerPlayer
  4. Si hay alts, anota las cuentas
  5. Aplica ban: /ban HackerPlayer Fly Fly hacks confirmado con video
  6. Si tiene alts: /ban AltAccount AltAccount
  7. Sube evidencias a Discord en el hilo del webhook

🔄 Escalera Fly: 1ra = 10d | 2da = 45d | 3ra = Permanente

Ejemplo 5: Helper - Investigar Multicuentas

Situación: Sospechas de multicuentas

Pasos:
  1. Verifica dupeip: /dupeip SuspectPlayer
  2. Revisa historial de cada cuenta: /history Account1, /history Account2
  3. Verifica ubicación: /geoip SuspectPlayer
  4. Revisa IPs: /iphistory SuspectPlayer
  5. Documenta todo (capturas de /dupeip y /history)
  6. Reporta a Moderator con evidencias

📝 Helper investiga, Moderator/Admin decide el castigo.

Ejemplo 6: Moderator - Griefing con Evidencia

Situación: Jugador reporta griefing con evidencia

Pasos:
  1. Teleportate al lugar: /tp VictimaPlayer
  2. Verifica la destrucción (usa CoreProtect si disponible)
  3. Revisa inventario del griefer: /invsee GrieferPlayer
  4. Verifica historial: /history GrieferPlayer
  5. Toma capturas de la evidencia
  6. Aplica ban: /ban GrieferPlayer Griefing Destruyó construcción de VictimaPlayer
  7. Sube evidencias a Discord (capturas + logs)

🔄 Escalera Griefing: 1ra = 3d | 2da = 14d | 3ra = 60d | 4ta = Permanente

Ejemplo 7: Moderator - Remover Castigo por Error

Situación: Te equivocaste de template o jugador

Pasos:
  1. Verifica el castigo: /history PlayerIncorrecto
  2. Contacta a Admin inmediatamente
  3. Explica el error detalladamente
  4. Si Admin autoriza, remueve: /unban PlayerIncorrecto Error: template incorrecto, debía ser warn
  5. Documenta el error en Discord staff
  6. Discúlpate con el jugador afectado
  7. Aplica el castigo correcto si corresponde

⚠️ SIEMPRE consulta con Admin antes de remover castigos.

Ejemplo 8: Helper - Usar Freeze para Investigar

Situación: Necesitas hablar con un jugador sobre su comportamiento

Pasos:
  1. Activa vanish: /vanish
  2. Teleportate: /tp PlayerTarget
  3. Congela: /freeze PlayerTarget
  4. Envíale mensaje privado: /msg PlayerTarget Hola, necesito hablar contigo sobre...
  5. Explica el problema con calma
  6. Si coopera y entiende: /freeze PlayerTarget (descongela)
  7. Si no coopera: aplica warn/kick según corresponda

💬 Freeze es útil para evitar que escapen mientras investigas.

❓ FAQ Rápido

Pregunta Respuesta
¿Por qué solo templates? Para consistencia y evitar abusos. El sistema aplica escaleras automáticas.
¿Me equivoqué de template? Contacta a un admin. Ellos pueden modificar el castigo.
¿Olvidé subir evidencia? Ve a #sanciones-evidencias, busca el webhook y crea el hilo ahora.
¿No estoy seguro si banear? NO BANEES. Consulta con admins primero.
¿Puedo remover castigos? (Mod) Sí, pero SOLO con autorización de Admin. Siempre proporciona razón.
¿Helper puede banear/mutear? NO. Helper solo puede /warn y /kick. Para ban/mute, contactar Moderator.
¿Jugador me insulta? Mantén calma. Captura, aplica castigo correspondiente, reporta.
¿Qué hago si encuentro hacker? Helper: Toma evidencias, congela, reporta a Mod. Mod: Verifica evidencias y banea.
